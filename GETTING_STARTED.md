# Minecraft.ai - Getting Started

## 📋 Требования

- Python 3.8+
- llama.cpp сервер (запущен на `localhost:8000`)
- Minecraft сервер (опционально для полной интеграции)

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск llama.cpp сервера

В отдельном терминале:

```bash
# Из директории llama.cpp
./server -m models/mistral-7b-instruct.gguf -ngl 35 --host localhost --port 8000
```

Или если у вас уже есть сервер:
```bash
# Просто убедитесь, что он доступен на localhost:8000
```

### 3. Запуск агента

```bash
python main.py
```

## 🎯 Как это работает

1. **Агент просыпается** - не знает правил
2. **Наблюдает мир** - получает состояние (позиция, здоровье, окружение)
3. **Думает** - через llama.cpp генерирует план
4. **Действует** - выполняет действие (копает, движется, ест)
5. **Рефлексирует** - записывает опыт в память
6. **Учится** - использует накопленный опыт для лучших решений

## 🧠 Ключевая фишка: Встроенная мотивация

Агент не следует скриптам. У него есть **собственные желания**:
- Выживать (избежать смерти)
- Понимать (любопытство)
- Создавать (созидание)
- Расти (самосовершенствование)

Все это в `ai/system_prompt.txt` - это не внешние награды, а встроенная в агента психология.

## 📊 Режим симуляции

Сейчас проект работает в **симуляционном режиме**:
- Нет реального Minecraft сервера - состояние генерируется
- llama.cpp используется для настоящих рассуждений
- Память сохраняется для долгосрочного обучения

Это позволяет тестировать логику без полноценного Minecraft сервера.

## 💡 Идеи для экспериментов

1. **Эволюция целей**: Смотри как агент переходит от "выжить" к "создать красивый дом"
2. **Долгосрочная память**: Сохрани сессию и посмотри как агент вспоминает прошлое
3. **Экстернализм**: Позволь агенту писать заметки сам себе (не скрипт - словесные напоминания)
4. **Творчество**: Позволь агенту выбирать где строить, какого цвета дом он хочет

## ❓ Часто задаваемые вопросы

**Q: Это просто случайные действия?**
A: Нет. ИИ через llama.cpp реально думает о ситуации и принимает решения на основе логики.

**Q: Нужен ли мне Minecraft?**
A: Пока нет. Сейчас это симуляция. Позже можно подключить реальный Minecraft.

**Q: Агент учится?**
A: Да. Память сохраняется между сессиями, и agent использует прошлый опыт для лучших решений.

## 🐛 Troubleshooting

### llama.cpp connection failed
```
⚠️  Warning: llama.cpp not available (will use dummy mode)
```
Убедитесь, что llama.cpp сервер запущен на `localhost:8000`

### Memory file not found
Создается автоматически при первом запуске в `memory/agent_log.json`
