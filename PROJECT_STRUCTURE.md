# Minecraft.ai Project Structure

## 📚 Быстрый навигатор

### 🎯 Начинающему разработчику
1. Прочитай [README.md](README.md) - обзор проекта
2. Посмотри [GETTING_STARTED.md](GETTING_STARTED.md) - как запустить
3. Посмотри [main.py](main.py) - точка входа

### 🧠 Как работает ИИ
- [ai/system_prompt.txt](ai/system_prompt.txt) - "мозг" агента (как он думает)
- [core/agent.py](core/agent.py) - основной цикл агента (думать → действовать → учиться)
- [core/llama_integration.py](core/llama_integration.py) - использование llama.cpp

### 🎮 Интеграция с Minecraft
- [minecraft_api/minecraft_client.py](minecraft_api/minecraft_client.py) - подключение к игре
- [config/config.json](config/config.json) - настройки подключения

### 📝 Память агента
- [memory/agent_log.json](memory/agent_log.json) - все что узнал агент

### 📋 Планы и идеи
- [ROADMAP.md](ROADMAP.md) - план развития проекта

## 💡 Ключевые концепции

### Встроенная мотивация (не скрипты)
```
❌ Плохо: if (health < 5) agent.eat_food()
✅ Хорошо: агент сам понимает что нужна еда
```

### Долгосрочная память (не забывает)
```
День 1: "Я чуть не утонул в воде"
День 100: "О, вода! Я помню что это опасно"
```

### Самостоятельное целеполагание (не команды)
```
❌ Плохо: agent.build_house_at(100, 64, -200)
✅ Хорошо: агент сам выбирает где строить дом
```

## 🚀 Как это запустить прямо сейчас

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Убедиться что llama.cpp сервер работает
# (если нет - просто запустится в симуляции)

# 3. Запустить!
python main.py
```
