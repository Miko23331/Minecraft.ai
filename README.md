# Minecraft.ai - ИИ агент с встроенной мотивацией

Проект по созданию ИИ агента в Minecraft, который учится **сам**, имеет собственные цели и желания.

## Структура проекта

- `core/` - Основной игровой движок агента
- `ai/` - Промпты и логика ИИ
- `memory/` - Система долгосрочной памяти агента
- `minecraft_api/` - Интеграция с Minecraft
- `config/` - Конфигурация llama.cpp и параметры

## Принцип работы

1. ИИ получает состояние Minecraft (позиция, здоровье, инвентарь, окружение)
2. ИИ думает и генерирует план действий на основе своих целей
3. ИИ совершает действие (движение, крафт, копание)
4. Результат записывается в память и влияет на будущие решения
5. **Никаких внешних скриптов** - только встроенная мотивация
